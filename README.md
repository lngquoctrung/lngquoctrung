- **Key Responsibilities (Team Leader):**

  - Processed and prepared bilingual dataset with appropriate preprocessing techniques including tokenization, sequence padding, and special token handling[cite: 17].
  - Implemented and trained a BiLSTM encoder-decoder architecture with multi-head attention to capture contextual information in both languages[cite: 18].
  - Built an inference system that enables real-time translation of English sentences to Vietnamese with proper handling of unknown words[cite: 19].
- **Source Code:** [machine-translation-bilstm-lstm](https://github.com/lngquoctrung/machine-translation-bilstm-lstm) [cite: 22]

### **3. Diabetes Prediction with BRFSS 2021 dataset**

- **Description:** A machine learning system that predicts the likelihood of diabetes using health indicators from the CDC BRFSS 2021 data and tested on BRFSS 2023 data[cite: 23].
- **Technologies:** Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn[cite: 24].
- **Key Responsibilities:**
  - Collected and preprocessed the 2021 BRFSS dataset to classify individuals as non-diabetic, pre-diabetic, or diabetic[cite: 25].
  - Performed exploratory data analysis and feature engineering, using visualization and correlation analysis to identify relevant health indicators[cite: 26].
  - Trained and fine-tuned classification models (Random Forest, K-Nearest Neighbors, Multi-layer Perceptron) with hyperparameter tuning via GridSearchCV[cite: 27].
  - Applied oversampling and undersampling techniques (SMOTE, ADASYN, TomekLinks) to address data imbalance[cite: 28].
  - Evaluated the generalization of the trained 2021 models on 2023 data to assess model robustness[cite: 30].
- **Source Code:** [diabetes-prediction-machine-learning](https://github.com/lngquoctrung/diabetes-prediction-machine-learning) [cite: 31]
